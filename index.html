Task Walkthrough Guide

  1. Extract Archive: Right-click Unorganized_Project_Data.zip in Downloads and click Extract all.
  2. Create Root Folder: Click New folder in Downloads.
  3. Rename Root Folder: Rename New folder to BCIS1110_Project_Lastname.
  4. Create Subfolders: Double-click inside your root folder and create 3 subfolders: Financial_Reports, Media_Assets, and Client_Docs.
  5. Move Media Files: Navigate back to Unorganized_Project_Data. Multi-select campus_logo.png, chart_q3.jpg, and banner_event.png. Cut & Paste into Media_Assets.
  6. Move Spreadsheets: Search .xlsx or select all 4 spreadsheet files (`budget_2026.xlsx`, `financial_audit.xlsx`, `expense_q1.xlsx`, `payroll_summary.xlsx`). Cut & Paste into Financial_Reports.
  7. Move Client Docs: Select client_contract.docx, client_proposal.docx, and meeting_minutes.docx and Cut & Paste into Client_Docs.
  8. Standardize File Name: Inside Client_Docs, rename client_proposal.docx to Client_Proposal_Final.docx.
  9. Purge System Caches: In Unorganized_Project_Data, delete clutter files (temp_cache.tmp, backup_old.bak, draft_trash.txt).
  10. Archive Project: In Downloads, right-click BCIS1110_Project_Lastname and click Compress to Zip file.
