Common Student Mistakes

  • Forgeting to extract Data Files Folder.zip before working inside it.
  • Leaving the zipper icon on the uncompressed folder.
  • Not moving the final zipped file into the File Management folder.
  • Deleting files before separating the job document files.

Assignment Instructions

Target Directory: Downloads > Managing Files Lastname
  1. Extract Data Files: Click Extract All on Data Files Folder.zip in Downloads.
  2. Rename Root Folder: Change Resources - Data Files to Managing Files Lastname.
  3. Create Subfolders: Create 3 new folders: Pictures, Files, and Aspen Manual.
  4. Rename Trip Pics: Rename Trip Pics to Trip Pictures.
  5. Move Photos: Hold Ctrl, select City Hall, Community Center, Runner. Cut & Paste into Pictures.
  6. Move PDFs: Drag the 4 PDF files into Aspen Manual.
  7. Shift Select Job Files: Type job in search. Click the first, hold Shift, click the last (5 files total). Cut & Paste into Files.
  8. Delete Loose Files: Drag-select or select all 8 remaining loose files in the main folder and press Delete.
  9. Rename Promotion: Inside Aspen Manual, rename Promotion to Promotion Plan.
  10. Compress to Zip: Right-click Managing Files Lastname and select Compress to Zip file. Move it to File Management.
